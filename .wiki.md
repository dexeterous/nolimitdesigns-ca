# Project Summary
The project, **NoLimit Designs**, is a subscription-based service providing unlimited graphic and video design solutions to businesses. Clients can submit an unlimited number of design requests fulfilled by a dedicated team of professional designers. This model allows companies to access high-quality creative assets efficiently, enhancing their marketing efforts without the overhead of in-house staff. The service offerings include various subscription plans detailed on the **Pricing page** and now features a comprehensive blog section offering design tips and insights.

# Project Module Description
The project is structured into several functional modules, including:
- **Homepage**: Features sections such as hero, benefits, updated pricing details, testimonials, and FAQs.
- **Components**: Reusable UI components like headers, footers, buttons, and cards.
- **Styling**: CSS and Tailwind configurations to maintain consistent design across the application.
- **SEO Scripts**: Tools for generating sitemaps and converting blog content to HTML for better search engine visibility.
- **Services Page**: A dedicated page for "NoLimit Graphics" detailing services, pricing, and features.
- **Video Design Services Page**: A page outlining video design offerings, including various types of video services and pricing.
- **Pricing Page**: A page detailing subscription plans, including features, pricing, and FAQs.
- **Blog Section**: A new module featuring a blog listing page and individual blog post pages, providing valuable insights and design tips.

# File Description Inventory
- **README.md**: Contains project setup instructions.
- **index.html**: Entry point for the web application.
- **App.tsx**: Main component that sets up routing and layout.
- **components/**: Contains reusable UI components such as SiteHeader and SiteFooter.
- **hooks/**: Custom React hooks for managing state and side effects.
- **pages/**: Contains individual page components including the homepage, GraphicDesignServices, VideoDesignServices, Pricing, Blog, and BlogPost.
- **data/**: Contains blog data and types.
- **lib/**: Helper functions and API integration.
- **public/**: Directory for static files including images and stylesheets.

# Technology Stack
- **Frontend**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: React Query
- **Icons**: Remix Icon

# Usage
To set up the project:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using the package manager:
   ```
   pnpm install
   ```
4. Build the project:
   ```
   pnpm run build
   ```
5. Run the application:
   ```
   pnpm run start
   ```
