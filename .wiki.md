# Project Summary
Nolimit Designs is a professional web development agency based in Edmonton, specializing in creating SEO-friendly websites for various industries, including healthcare and real estate. The project aims to enhance online visibility, drive traffic, and generate leads through a robust digital presence, offering services from web design to digital marketing. Recently, the project has improved email handling by integrating the Resend API, allowing for efficient management of contact form submissions.

# Project Module Description
The project consists of various functional modules, including:
- **Homepage**: Showcases agency services and capabilities with an updated hero image and includes Google site verification.
- **Industry Portfolio Pages**: Dynamic pages presenting tailored solutions for different industries.
- **Service Pages**: Detailed descriptions of web development, SEO, e-commerce, and more.
- **Location-Based SEO Pages**: Targeted pages for specific regions around Edmonton.
- **Contact Page**: Lead generation with forms, chat, and booking options, featuring an updated phone number and integrated Resend API for email notifications.
- **Blog**: Articles focused on web design and development.
- **Demo Mini-Websites**: Static HTML representations for healthcare, real estate, and restaurant industries.

# File Description Inventory
- **App.tsx**: Main application file that sets up routing and context providers.
- **SiteHeader.tsx**: Component for the website's navigation header.
- **SiteFooter.tsx**: Component for the website's footer with updated contact information.
- **Index.tsx**: Homepage of the web application showcasing services and industries with a new hero image and Google verification meta tag.
- **IndustryPortfolio.tsx**: Displays portfolio items related to a selected industry.
- **ServicePage.tsx**: Detailed information about a specific service offered.
- **LocationPage.tsx**: Information tailored to specific service locations.
- **Contact.tsx**: Contact form and lead generation interface with updated phone number and Resend API integration for email notifications.
- **About.tsx**: Information about the agency and its values with updated phone number.
- **Blog.tsx**: Displays blog posts related to web design and development.
- **blogPosts.ts**: Contains data for the blog posts.
- **googled0bc1acca73197cc.html**: Google verification file for analytics verification.
- **index.html**: Main HTML file for the web application, includes Google site verification meta tag.
- **resend_email.py**: Backend service for sending emails via the Resend API, updated to handle email sending restrictions and configuration.
- **resend_email.py (router)**: Exposes an API endpoint for sending contact form submissions via email, now with improved error handling.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, FastAPI
- **Database**: PostgreSQL (if applicable)
- **Deployment**: Vercel or similar platforms
- **Email Service**: Resend API

# Usage
To set up and run the project:
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Start the development server:
   ```bash
   pnpm start
   ```
