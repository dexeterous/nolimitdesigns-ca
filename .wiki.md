# Project Summary
Nolimit Designs is a professional web development agency based in Edmonton, transitioning from a design subscription service to a comprehensive provider. The project focuses on creating SEO-friendly websites tailored for over 20 industries, including healthcare, real estate, and more. It aims to enhance online visibility and generate leads through a robust digital presence.

# Project Module Description
The project consists of various functional modules, including:
- **Homepage**: Redesign to reflect agency services and capabilities.
- **Industry Portfolio Pages**: Dynamic pages for 20 industries showcasing tailored solutions.
- **Service Pages**: Detailed descriptions of web development, SEO, e-commerce, and more.
- **Location-Based SEO Pages**: Targeted pages for various regions around Edmonton.
- **Contact Page**: Lead generation with forms, chat, and booking options.
- **Demo Mini-Websites**: Static HTML representations for healthcare, real estate, and restaurant industries.

# File Description Inventory
- **App.tsx**: Main application file that sets up routing and context providers.
- **SiteHeader.tsx**: Component for the website's navigation header.
- **SiteFooter.tsx**: Component for the website's footer with links and contact info.
- **Index.tsx**: Homepage of the web application showcasing services and industries.
- **IndustryPortfolio.tsx**: Displays portfolio items related to a selected industry.
- **ServicePage.tsx**: Detailed information about a specific service offered.
- **LocationPage.tsx**: Information tailored to specific service locations.
- **Contact.tsx**: Contact form and lead generation interface.
- **About.tsx**: Information about the agency and its values.

# Usage
To set up and run the project:
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Start the development server:
   ```bash
   pnpm start
   ```
