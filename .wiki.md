# Project Summary
**NoLimit Designs** is a subscription-based service offering unlimited graphic and video design solutions tailored for businesses. Clients can submit an unlimited number of design requests, which are fulfilled by a dedicated team of professional designers. This model provides companies with high-quality creative assets efficiently, enhancing their marketing efforts without the cost of maintaining in-house staff. The service includes various subscription plans, a comprehensive blog section for design tips, and now features a robust client and admin dashboard for improved user experience.

# Project Module Description
The project is structured into several functional modules, including:
- **Homepage**: Showcases hero sections, benefits, updated pricing details, testimonials, and FAQs.
- **Components**: Reusable UI components like headers, footers, buttons, and cards.
- **Styling**: CSS and Tailwind configurations to maintain consistent design across the application.
- **SEO Scripts**: Tools for generating sitemaps and converting blog content to HTML for better search engine visibility.
- **Services Page**: Details services, pricing, and features of "NoLimit Graphics."
- **Video Design Services Page**: Outlines video design offerings, including various types of video services and pricing.
- **Pricing Page**: Provides details on subscription plans, features, pricing, and FAQs.
- **Blog Section**: Features a blog listing page and individual blog posts offering insights and design tips.
- **Client Dashboard**: SaaS-style dashboard for clients with stats, recent requests, and activity feeds.
- **Admin Dashboard**: Overview for admins with metrics and designer workload visualization.
- **Request Management**: Client and admin pages for managing design requests, including submission forms and request lists.

# File Description Inventory
- **README.md**: Contains project setup instructions.
- **index.html**: Entry point for the web application.
- **App.tsx**: Main component that sets up routing and layout.
- **components/**: Contains reusable UI components such as SiteHeader, SiteFooter, and DashboardLayout.
- **hooks/**: Custom React hooks for managing state and side effects.
- **pages/**: Contains individual page components including homepage, GraphicDesignServices, VideoDesignServices, Pricing, Blog, BlogPost, Registration, Login, Client Dashboard, and Admin Dashboard.
- **data/**: Contains blog data and types.
- **lib/**: Helper functions and API integration.
- **public/**: Directory for static files including images and stylesheets.

# Technology Stack
- **Frontend**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: React Query
- **Icons**: Remix Icon

# Usage
To set up the project:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using the package manager:
   ```
   pnpm install
   ```
4. Build the project:
   ```
   pnpm run build
   ```
5. Run the application:
   ```
   pnpm run start
   ```
