# Project Summary
Nolimit360 is an advanced design management platform that optimizes the submission, tracking, and management of design requests. It fosters collaboration between clients and designers with real-time updates and an intuitive interface, enhancing design workflow efficiency for businesses. Recent enhancements include a drag-and-drop Kanban board, an email notification system, and various bug fixes to ensure smooth design request submissions.

# Project Module Description
- **Backend**: Handles database interactions, authentication, and CRUD operations for design requests. Features include brand management, notifications, and design file handling. Recent updates have resolved validation issues in API schemas and enabled automatic synchronization of new database columns.
- **Frontend**: Offers a dynamic user interface for clients and admins to manage design requests, view statistics, and collaborate with the design team. Enhancements include an improved Kanban board, detailed request views, and a new notification settings page.

# File Description Inventory
- **app/backend/main.py**: Entry point for the backend application.
- **app/backend/routers/**: Contains API routes for managing requests, brands, notifications, and user preferences.
- **app/backend/models/**: Defines the database schema for requests, users, brands, and notifications.
- **app/frontend/src/**: Contains the frontend application code, including pages and components with recent improvements.
- **app/frontend/public/**: Stores public assets such as images and stylesheets.
- **app/frontend/src/todo.md**: Document outlining the enhancement plan and tasks.
- **app/frontend/src/contexts/AuthContext.tsx**: Manages authentication context and admin role checks.

# Technology Stack
- **Backend**: FastAPI, SQLAlchemy, PostgreSQL
- **Frontend**: React, TypeScript, Tailwind CSS, Vite

   - For the frontend: Navigate to `app/frontend` and run the dependency manager to install required packages.

2. **Build the Application**:
   - For the backend: Use the command to build the backend application.
   - For the frontend: Use the command to build the frontend application.

3. **Run the Application**:
