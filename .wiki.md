# Project Summary
The project, **NoLimit Designs**, is a subscription-based service that provides unlimited graphic and video design solutions to businesses. It streamlines the creative process by allowing clients to submit an unlimited number of design requests, fulfilled by a dedicated team of professional designers. This model enables companies to access high-quality creative assets without the overhead of hiring in-house staff, thus enhancing their marketing efforts efficiently. The service offerings have been expanded with the introduction of a **Pricing page**, which details various subscription plans.

# Project Module Description
The project is structured into several functional modules, including:
- **Homepage**: Features sections such as hero, benefits, pricing, testimonials, and FAQs.
- **Components**: Reusable UI components like headers, footers, buttons, and cards.
- **Styling**: CSS and Tailwind configurations to maintain consistent design across the application.
- **SEO Scripts**: Tools for generating sitemaps and converting blog content to HTML for better search engine visibility.
- **Services Page**: A dedicated page for "NoLimit Graphics" detailing services, pricing, and features.
- **Video Design Services Page**: A page that outlines video design offerings, including various types of video services and pricing.
- **Pricing Page**: A new page that details subscription plans, including features, pricing, and FAQs.

# File Description Inventory
- **README.md**: Contains project setup instructions.
- **index.html**: Entry point for the web application.
- **App.tsx**: Main component that sets up routing and layout, including new routing for the Pricing page.
- **components/**: Contains reusable UI components such as SiteHeader and SiteFooter.
- **hooks/**: Custom React hooks for managing state and side effects.
- **pages/**: Contains individual page components like the homepage, GraphicDesignServices, VideoDesignServices, and the new Pricing page.
- **lib/**: Helper functions and API integration.
- **public/**: Directory for static files including images and stylesheets.

# Technology Stack
- **Frontend**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: React Query
- **Icons**: Remix Icon

# Usage
To set up the project:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using the package manager:
   ```
   pnpm install
   ```
4. Build the project:
   ```
   pnpm run build
   ```
5. Run the application:
   ```
   pnpm run start
   ```
