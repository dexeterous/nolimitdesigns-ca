Build Nolimit360 Design Management Platform (Client + Admin)
PROJECT OVERVIEW

Build Nolimit360, the internal design request management platform for Nolimit Designs (nolimitdesigns.ca).

This is a SaaS platform used by two user groups:

Clients
Submit design requests, review work, collaborate with designers.
Admin / Designers (internal team)
Manage requests, assign designers, upload deliverables, track productivity.

The system must feel like a modern SaaS dashboard similar to Linear, ClickUp, or Notion.

CRITICAL DESIGN RULE

Reuse the existing Nolimit UI design system and components wherever possible.

This includes:

Typography scale
Color tokens
Button styles
Input components
Cards
Badges
Avatars
Icons
Spacing system
Modals
Form elements

Do NOT introduce a new visual system.

The dashboard must feel like a natural extension of the Nolimit marketing website.

PLATFORM STRUCTURE

The application has two separate dashboard environments.

/client-dashboard
/admin-dashboard

Each uses the same base design system but different permissions.
